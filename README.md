# connect-invest-talk

A website that connects business owners with investors based on eligibility criteria for verification on both parties and can talk to each other via chat on the platform. Sign in is required, and this website is SaaS-enabled marketplace.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/connect-invest-talk.git
cd connect-invest-talk
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
